Hello, world! 1. 人工知能とは何か 人工知能とは、いわば人の知能を人工的に作り出したものである。しかし、まだ人の脳のすべてを完璧に再現したものはない。そのため、脳の一部の機能を模したものが、現在AIと呼ばれているものである。 AIが行うものとしては ・ 音声認識 : 人が話した内容を理解させる ・ 画像認識 : 画像に何が映っているかなどを認識させる ・ 感性処理 : 人間の感覚を実現させる ・ 機械学習 : データの集合から、ある一定の法則を見つけ出す ・ 自然言語処理 : 文章などの言語の内容を理解させる ・ 情報検索 : データの中から検索を行う ・ 推論 : 今までのデータから次の答えを導きだす ・ マルチエージェント : 複数のユニットが集まり、複雑な問題を解く などが挙げられる。 このようにその幅は多岐にわたる。 今回はその中でも画像認識の領域に着目して進めていきたい。 2. 画像領域における人工知能 画像認識における人工知能は、主に人の視覚を模したものであり、「Deep learning」と呼ばれている。 Deep learning とはニューラルネットワークを用いた人工知能の構築技術の総称とされており、脳(神経細胞)の動きを模した学習アルゴリズムである。 ここで、ニューラルネットワークについて説明しておく。人の脳細胞はニューロンという神経細胞の集まりでできており、それらが相互に情報をやり取りすることで活動を行っている。このニューロン一つでは単純な動きしかしておらず、この動きを模したものとしては1960年ごろに単純パーセプトロンという名前で実現されている。しかし、先にも書いたように、一つでは単純な動きしかできないので、複雑な問題に対しては適用できないものであった。 そこで考えられたのがこれらを複数組み合わせ、層状にした多層パーセプトロン(ニューラルネットワーク)であった。人の脳の視覚野も層状の構造を持っていることが知られており、より人の脳に近いものとなった。この学習には誤差逆伝搬法という方法が用いられたが、この層が多くなるとうまく学習できないことがわかった。 そこで現れたのがDeep learning と呼ばれる方法であり2006年Hintonらによって提唱された。これはこれまでの学習における問題点を改善するとともに、近年のインターネット等による学習するためのデータ数の増加や計算機の高速化により実現することができたと考えられている。 Deep learning ではいくつかの少ない層で学習を行い、その後それらを組み合わせることで問題を解決している。しかし、この層が増えると学習するための時間的、資源的(学習データやそれらのパラメータ)なコストが大きくなり、簡単に扱えるものでない。そのため、様々な学習方法が考えられており、最近ではオープンソースでDeep learningを行えるようなものが公開されている。 3. アルゴリズム Deep learning のアルゴリズムは様々あるが、今回は Convolutional Neural Network (CNN) と呼ばれる方法に関して見ていきたい。 これはLeCunらが提唱した手法であり、一つの視細胞が見ている領域(受容野)の畳み込みと集約(プーリング)を一つの層として、これらを複数組み合わせて、最終的に識別したいものの数と同じ数のニューロン数にしていくものである。 畳み込みでは、画像の一部分から、のような直線などを抽出するような複数のフィルタを掛け、それらをずらしながら画像全体に適用していく。 こうして得られた情報を集約し、ある意味ぼかすことにより、より画像として特徴的な部分を抽出していく。この操作を繰り返すことにより、複雑で画像の特徴を表したものが出来上がる。 実際の学習では、出来たものに対して答え合わせをし、その正解との誤差を用いて再び学習を行い、よりよいものに仕上げていく。
